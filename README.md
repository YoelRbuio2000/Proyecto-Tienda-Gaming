# Aeternum Tech - Hardware Store

Este es un proyecto de práctica desarrollado como estudiante de programación, enfocado en crear una experiencia de usuario fluida para una tienda de hardware de alto rendimiento.

## Características
- **Carga Dinámica:** Los productos se cargan desde un archivo `JSON` externo simulando una base de datos.
- **Carrito de Compras:** Sistema funcional de agregar/quitar productos con cálculo de total en tiempo real.
- **Interfaz Responsiva:** Diseño oscuro (dark mode) optimizado para entusiastas del hardware.
- **Event Handling:** Manejo avanzado de eventos en el DOM para una navegación sin recargas.

## 🛠️ Tecnologías Usadas
- **HTML5** & **CSS3** (Variables y Flexbox).
- **JavaScript (ES6+)**: Fetch API, Async/Await, y manipulación del DOM.

## 📂 Estructura del Proyecto
- `/js`: Lógica de la tienda (`productos.js`).
- `/assets`: Imágenes de componentes de alta calidad.
- `/style`: Hojas de estilo personalizadas.
